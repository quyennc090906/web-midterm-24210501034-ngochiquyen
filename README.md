# 2. Mô tả dự án

EduMini là website giới thiệu và đăng ký các khóa học lập trình.

Người dùng có thể:

- Xem danh sách khóa học
- Tìm kiếm khóa học
- Lọc theo danh mục
- Xem chi tiết khóa học
- Đăng ký tham gia
- Quản lý danh sách đăng ký

- LocalStorage

# 4. Chức năng chính


## Trang chủ

- Giới thiệu website


## Danh sách khóa học

- Hiển thị 8 khóa học

- Tìm kiếm

- Lọc dữ liệu

- Xem chi tiết bằng Modal


## Đăng ký

- Form đăng ký

- Kiểm tra dữ liệu nhập


## Quản lý đăng ký

- Lưu bằng LocalStorage

- Hiển thị danh sách

- Xóa đăng ký
