# AI Usage Report


## 1. Em có sử dụng AI không?


Có



---

## 2. AI hỗ trợ phần nào?


AI hỗ trợ:

- Gợi ý cấu trúc thư mục dự án

- Hướng dẫn viết HTML/CSS

- Giải thích JavaScript

- Hỗ trợ sửa lỗi code

- Tạo dữ liệu khóa học mẫu



---

## 3. Prompt đã sử dụng


Ví dụ:


"Viết website đăng ký khóa học bằng HTML CSS JavaScript Bootstrap"


"Giải thích code JavaScript dễ hiểu"



---

## 4. Em đã chỉnh sửa gì sau khi AI sinh code?


- Thay đổi giao diện

- Chỉnh lại màu sắc

- Thêm dữ liệu khóa học

- Điều chỉnh form đăng ký

- Sửa lỗi đường dẫn file



---

## 5. Phần nào em tự viết?


- Tạo cấu trúc website

- Chỉnh sửa HTML

- Chỉnh CSS

- Kiểm tra hoạt động chương trình



---

## 6. Em học được gì?


Sau khi thực hiện dự án, em hiểu được:

- Cách xây dựng website frontend

- Cách sử dụng Bootstrap

- Cách thao tác DOM bằng JavaScript

- Cách lưu dữ liệu bằng LocalStorage

- Cách kiểm tra dữ liệu form
